Unidad I

Introducción a los Microprocesadores y Microcontroladores

Introducción

La arquitectura y organización de computadoras estudia el diseño interno de los sistemas computacionales, permitiendo comprender la interacción entre hardware y software para ejecutar instrucciones y procesar datos.

1.1 Conceptos de Arquitectura y Organización

1.1.1 Arquitectura

La arquitectura de computadoras describe las características visibles para el programador y usuario.

  • Conjunto de instrucciones
  • Formato de datos
  • Registros
  • Manejo de memoria

1.1.1.2 Organización

La organización se refiere a cómo se implementan físicamente los componentes del sistema.

  • CPU
  • Memoria
  • Buses
  • Dispositivos de Entrada/Salida

1.1.1.2.1 Estructura

La estructura representa la forma en que los componentes internos están organizados.

  • CPU
  • Memoria principal
  • Periféricos
  • Buses

1.1.1.2.2 Función

Las funciones principales de una computadora son:

  • Procesamiento de datos
  • Almacenamiento
  • Transferencia de información
  • Control de operaciones

1.1.2 Estructura de la Computadora

Procesamiento

El procesamiento se realiza en la CPU.

  • ALU
  • Unidad de Control
  • Registros

Sistema de Memoria

Almacena datos e instrucciones necesarias.

  • RAM
  • ROM
  • Caché
  • Disco Duro

Entrada - Salida

Permite la comunicación entre usuario y computadora.

  • Teclado
  • Mouse
  • Monitor
  • Impresora

1.2 Diferencia entre Microprocesador y Microcontrolador

Microprocesador

Circuito integrado encargado de ejecutar operaciones.

Ejemplos: Intel Core i7 y AMD Ryzen.

Microcontrolador

Circuito integrado con CPU, memoria y puertos E/S.

Ejemplos: Arduino, PIC y ESP32.

Comparación

Microprocesador Microcontrolador
Necesita memoria externa Integra memoria
Mayor potencia Menor consumo energético
Usado en computadoras Usado en sistemas embebidos
Mayor costo Más económico
