Tutorial de Arduino

Aprende los fundamentos de Arduino y realiza prácticas básicas de programación y electrónica.

1. Introducción a Arduino

Arduino es una plataforma de hardware y software libre utilizada para desarrollar proyectos electrónicos interactivos. Se basa en microcontroladores programables y un entorno de desarrollo sencillo.

Es ampliamente utilizado en proyectos educativos, robótica, automatización y sistemas embebidos debido a su facilidad de uso y gran comunidad de soporte.

2. Componentes de una placa Arduino

Microcontrolador

Es el cerebro de la placa Arduino encargado de ejecutar las instrucciones del programa.

Entradas y Salidas

Permiten conectar sensores, LEDs, motores y otros dispositivos.

3. Instalación del IDE Arduino

  • Descargar Arduino IDE desde la página oficial.
  • Instalar el programa.
  • Seleccionar la placa Arduino.
  • Seleccionar el puerto COM.

4. Estructura básica de un programa Arduino

setup()

La función setup() se ejecuta una sola vez al iniciar el programa. Aquí se configuran los pines y dispositivos. 

void setup() {

    pinMode(13, OUTPUT);

}

loop()

La función loop() se ejecuta continuamente mientras Arduino esté encendido. 

void loop() {

    digitalWrite(13, HIGH);
    delay(1000);

    digitalWrite(13, LOW);
    delay(1000);

}

5. Prácticas de Arduino

5.1 LED Parpadeante

Este programa hace parpadear un LED conectado al pin 13. 

void setup() {

    pinMode(13, OUTPUT);

}

void loop() {

    digitalWrite(13, HIGH);
    delay(500);

    digitalWrite(13, LOW);
    delay(500);

}

5.2 Semáforo con LEDs

Simulación de un semáforo utilizando LEDs rojo, amarillo y verde. 

const int LED_ROJO=4;
const int LED_AMARILLO=3;
const int LED_VERDE=2;


void setup() {

    pinMode(LED_ROJO, OUTPUT);
    pinMode(LED_AMARILLO, OUTPUT);
    pinMode(LED_VERDE, OUTPUT);

}

void loop() {

    digitalWrite(LED_ROJO, HIGH);
    digitalWrite(LED_AMARILLO, LOW);
    digitalWrite(LED_VERDE, LOW);
    delay(2000);

    digitalWrite(LED_ROJO, LOW);
    digitalWrite(LED_AMARILLO, HIGH);
    digitalWrite(LED_VERDE, LOW);
    delay(2000);

    digitalWrite(LED_ROJO, LOW);
    digitalWrite(LED_AMARILLO, LOW);
    digitalWrite(LED_VERDE, HIGH);
    delay(2000);

}

5.3 Control de Temperatura

Uso de un sensor LM35 y un ventilador para mantener la temperatura. 

#define TEMPERATURE A0
#define MOTOR 5

void setup() {

    Serial.begin(9600);
    pinMode(MOTOR, OUTPUT);

}

int speed_decider(int temp) {
    if(temp<35)
        return 0;
    else if(temp>40)
        return 255;
    else
        return map(temp, 35, 40, 0, 255); 
}

void loop() {

    int temperature = analogRead(TEMPERATURE);
    temperature = map(temperature, 20, 358, -40, 125);
    Serial.println(speed_decider(temperature));
    analogWrite(MOTOR, speed_decider(temperature));
    delay(1000);
    
}
